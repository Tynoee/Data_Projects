# Manufacturing Quality Monitoring with Statistical Process Control (SPC)

## Project Overview
This project implements Statistical Process Control (SPC) to monitor manufacturing quality. Using SQL window functions and Python analysis, it calculates control limits (UCL/LCL) and flags out-of-spec parts requiring process adjustments.

## Tools & Technologies
- SQL  
- Python (Pandas, NumPy, Matplotlib)  
- Jupyter Notebook  

## Key Skills Demonstrated
- **SQL**: Window functions, nested queries, rolling calculations
- **Python**: Pandas, NumPy, Matplotlib, data transformation
- **Statistical Analysis**: 3-sigma control limits, process capability
- **Data Quality**: Alert detection, outlier identification

## Results
- Analyzed 420 manufactured units across 20 operators
- Identified **57 quality alerts (13.57% alert rate)**
- Pinpointed operators Op-4, Op-5, Op-16 for priority investigation

## Business Impact
This monitoring system enables real-time quality detection, reducing scrap and ensuring consistent product quality.
